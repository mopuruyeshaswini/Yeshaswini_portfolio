# 🌐 Personal Portfolio Website

A modern, responsive, and dynamic personal portfolio website built using HTML, CSS, and JavaScript. This portfolio showcases my education, technical skills, academic project, and contact information in a clean and professional design.

## ✨ Features

- Responsive design for all devices
- Modern and attractive user interface
- Animated typing effect
- Smooth scrolling navigation
- Interactive skill progress bars
- Education timeline
- Project showcase section
- Contact section with icons
- Clean and organized code

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts
- Font Awesome

## 📂 Sections

- Home
- About Me
- Education
- Skills
- Academic Project
- Contact

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in any modern web browser.
